#  Git Party

## Descrição do Projeto

O Git Party é uma aplicação Fullstack desenvolvido para o gerenciamento de eventos, shows e festivais.  
O sistema permite cadastrar, visualizar e excluir eventos através de uma interface moderna integrada com um backend em Node.js utilizando Prisma e MySQL.

O projeto foi desenvolvido com foco em integração entre frontend e backend, utilizando rotas REST para comunicação com a API.

---

# Funcionalidades

- Cadastro de eventos
- Listagem de eventos
- Exclusão individual de eventos
- Integração com API 
- Banco de dados MySQL
- Interface moderna responsiva
- Backend com Express
- Prisma
